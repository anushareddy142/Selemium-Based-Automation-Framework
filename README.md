# Selemium-Based-Automation-Framework
Framework for Application Automated Testing using Selenium WebDriver, produce ExtentReports in html format using TestNG, Excel based data inputs, JIRA integration for defect creation, maven as build library. Included framework or library classes for various reusable components.
